name donutShopModel
author: haanj

This models potential sales of various donut shop location using existing sale information. From the browser, the viewer can add additional donut shops to be added to the model. The RECALCULATE button will recalculate more sales, including any additional locations that have been added that session. The RESET button should return the state to the original one. SAVE and RECALL will save and recall the current state.
